About This App

Flask Demo Application

This is a simple Flask application deployed to AWS ECS using Terraform and GitHub Actions.

Technology Stack:

  • Flask - Python web framework
  • Docker - Containerization
  • AWS ECS - Container orchestration
  • Terraform - Infrastructure as Code
  • GitHub Actions - CI/CD pipeline

This demo application shows how to build a modern containerized application with automated deployment.
